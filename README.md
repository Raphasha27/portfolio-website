## 🚀 Deployment Documentation

### Prerequisites
- Node.js (Latest LTS)
- npm or yarn

### Local Development
```bash
# Clone the repository
git clone https://github.com/Raphasha27/Portfolio.git

# Navigate to project (React version is in v2)
cd portfolio/v2

# Install dependencies
npm install

# Start Dev Server
# Note: --host flag enabled for local network testing
npm run dev
```
