## Portfolio v2

High-fidelity, cinematic React/Vite implementation of the Kirov Dynamics professional branding ecosystem — featuring 3D neural network visualizations, interactive AI elements, and sovereign infrastructure concepts.

**Live:** [https://koketso-raphasha.vercel.app/](https://koketso-raphasha.vercel.app/)

## Features

- **Cinematic Dark-Luxury UI** — Tailwind CSS + Framer Motion with neon accents
- **Interactive Terminal** — Hacker-style terminal simulating AI interactions
- **Sovereign Intelligence** — Digital Twin AI assistant with context-aware responses
- **SEO Optimized** — Comprehensive metadata and JSON-LD structured data
- **Responsive** — Fluid layouts across desktop, tablet, and mobile

## Tech Stack

| Technology | Version |
